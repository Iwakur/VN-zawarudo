# Music README

This document is a practical guide for writing the main OST theme for the visual novel in a DAW. The goal is to create a memorable main melody that can be reused in different emotional versions across the project.

## Core Direction

The main theme should feel dramatic, emotional, and slightly mysterious. It should work as a full opening theme, but also be simple enough to adapt into quieter scene music.

Recommended emotional targets:

- Melancholic, but not hopeless
- Cinematic, but still intimate
- Serious, with a little warmth
- Easy to loop during menus or story scenes

## Recommended Key

Use **D minor** as the main key.

D minor is a strong fit because it feels dramatic, serious, and cinematic without becoming too dark. It also works well for piano, strings, pads, bells, and orchestral instruments.

Alternative keys:

- **A minor**: easier to write in, intimate, emotional, and natural for piano.
- **F major**: softer and more hopeful while still sharing many notes with D minor.
- **C major with minor chords**: simple, clean, and good for a less tragic mood.

Main recommendation:

```text
D minor
```

## Main Chord Progression

Start with this 8-bar loop:

```text
Dm - Bb - F - C
Dm - Gm - Bb - A
```

Roman numeral analysis in D minor:

```text
i  - VI - III - VII
i  - iv - VI  - V
```

This progression gives the theme emotion and resolution. The first four chords feel open and cinematic. The second four chords become more serious, with **A major** pulling strongly back to **D minor**.

## Other Chord Options

Use these depending on the mood of the scene.

### Dramatic Main Theme

```text
Dm - Bb - F - C
```

Use this for the main menu, opening, or general story scenes. It is emotional but not too dark.

### More Melancholic

```text
Dm - C - Bb - A
```

Use this when the scene needs more sadness or tension. The **A major** chord creates a strong return to D minor.

### Mysterious Visual Novel Mood

```text
Dm - Gm - Bb - A
```

Use this for quiet scenes, investigation scenes, inner thoughts, or moments that should feel uncertain.

### More Hopeful

```text
F - C - Dm - Bb
```

This uses the same harmonic world as D minor, but starts from F major. It feels more open, warm, and less tragic.

## Melody Notes

In D minor, build the main melody around these notes:

```text
D - F - G - A - C
```

These notes are stable and emotional. Use **Bb** when the melody needs more color. Use **C#** only near the end of a phrase when you want a strong pull back into **D**.

Simple main motif idea:

```text
D - F - A - G - F - E - D
```

Answer phrase:

```text
A - C - Bb - A - G - F - E - D
```

The melody should be singable. Avoid making it too busy. A strong OST theme usually has a clear phrase that the listener can remember after hearing it once or twice.

## Suggested Tempo And Meter

Recommended tempo:

```text
72-88 BPM
```

Recommended meter:

```text
4/4
```

Use a slower tempo for emotional piano versions. Use a slightly faster tempo if the track becomes an opening theme or trailer-style arrangement.

## DAW Arrangement

Start simple:

- Piano: main chords and melody sketch
- Soft strings or pad: sustained harmony
- Lead instrument: piano, violin, flute, music box, or soft synth lead
- Bass: root notes, very minimal
- Percussion: optional, low cinematic hits only if the scene needs scale

Do not over-arrange the first version. Write the melody and chords first, then add layers only when the theme already works on piano.

## Suggested 8-Bar Sketch

Use this as the first draft in the DAW:

```text
Bar 1: Dm
Bar 2: Bb
Bar 3: F
Bar 4: C
Bar 5: Dm
Bar 6: Gm
Bar 7: Bb
Bar 8: A
```

Possible melody shape:

```text
Bars 1-2: D - F - A - G - F
Bars 3-4: A - C - A - G - F - E
Bars 5-6: D - F - G - A - C
Bars 7-8: Bb - A - G - F - E - C# - D
```

The final **C# - D** movement should feel like the melody is coming home.

## Variations For The Game

Once the main theme works, create several versions from the same melody.

### Main Menu Version

- Piano melody
- Warm strings
- Soft pad
- Light low percussion
- Full 8-bar loop

### Sad Scene Version

- Solo piano or music box
- Slower tempo
- Fewer chords
- More silence between melody notes

Suggested progression:

```text
Dm - C - Bb - A
```

### Mystery Version

- Low piano
- Soft pad
- Sparse bell or reversed texture
- Avoid heavy drums

Suggested progression:

```text
Dm - Gm - Bb - A
```

### Hopeful Version

- Start in F major
- Use higher piano voicing
- Add soft strings later in the loop

Suggested progression:

```text
F - C - Dm - Bb
```

## Production Notes

- Keep the main melody in the mid range first, then move it up an octave for emotional lift.
- Let the bass follow root notes at first: D, Bb, F, C, D, G, Bb, A.
- Use reverb, but keep the melody clear.
- Avoid too many instruments playing the melody at once.
- If the track feels too sad, change the first chord to F major for a hopeful variation.
- If the track feels too flat, use A major instead of A minor at the end of the loop.
- If the track needs stronger tension, use C# before resolving to D.

## Quick Starting Point

For a main theme, write an 8-bar loop first using:

```text
Dm - Bb - F - C
Dm - Gm - Bb - A
```

That gives the OST both emotion and resolution.
